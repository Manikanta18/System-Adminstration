# Sytem-Adminstartion GUI for Linux

## Prerequisites

```
it should Linux machine
```
```
sudo apt-get update
```
Python3
```
sudo apt-get install python3
```
Tkinter
```
sudo apt-get install python python3-tk
```
Pandas
```
sudo apt-get install python3-pandas
```
matplotlib.pyplot
```
sudo apt-get install python3-plotly
```
Execution of GUI
```
Python3 GUI.py
```
